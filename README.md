## Introduction
This project focuses on **debugging and improving computer vision code** using tools like **PyTorch, PIL (Pillow), and Matplotlib**. It explores common issues faced in computer vision workflows and demonstrates how to identify, troubleshoot, and fix them effectively.  

The goal is to build a strong understanding of how to resolve errors in image processing, tensor operations, and model preparation.

In this project, I performed the following:  
- **Code Debugging**: Identified and fixed errors related to image loading, tensor shapes, and plotting issues.  
- **Tensor Corrections**: Resolved dimension mismatches and ensured correct formats such as (C x H x W).  
- **Visualization Fixes**: Corrected plotting mistakes in bar charts, image displays, and axis handling using Matplotlib.  
- **Data Handling**: Fixed issues in dataset loading, class distribution calculations, and preprocessing steps.  
- **Transformation Handling**: Applied correct image transformations using `torchvision.transforms` for model readiness.  
- **Error Analysis**: Interpreted error messages and debugged step-by-step to ensure smooth execution.
